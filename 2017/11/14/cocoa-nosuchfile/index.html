Cocoapods No such file or directory错误

Nov 14, 2017 | Hits

cocoapods diff: /../Podfile.lock: No such file or directory

pod install之后报的错误

解决方案 一条条的尝试，不是按顺序来的

第一种

1，关闭Xcode，重新执行pod install，之后再重新打开Xcode运行。

第二种

删除文件

删除生成的workspace

删除.lock

删除Pods

删除Build

删除DerivedData
然后执行pod install

第三种

3，选中PROJECT 然后选中Info tab 查看 Configurations 选择相应的配置文件

修改Pods脚本文件路径 查看xcodeproj 的project.pbxproj，查找”resources.sh”

然后重新打开workspace

第四种

移除cocopods后重新安装
1、打开终端，运行sudo gem install cocoapods-deintegrate安装快速解除项目cocopods依赖的库

2、安装成功后，cd到你项目的更目录运行pod deintegrate解除项目cocopods依赖

3、运行pod install,重新安装cocopods

话说第四种是最有效的。
